# **App Name**: Hola Mundo App

## Core Features:

- Display Text: Display the text 'Hola Mundo' on a blank page.

## Style Guidelines:

- Background color: Soft white (#F0F0F0) to provide a clean backdrop.
- Primary color: Deep Indigo (#3F51B5) for the 'Hola Mundo' text, providing a clear contrast and modern feel.
- Font: 'Inter', a grotesque-style sans-serif, suitable for body text due to its clean and readable design.
